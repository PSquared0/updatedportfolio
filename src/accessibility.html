Accessibility

Accessibility isn't something I added to this site as an afterthought. It's how I think about every product I work on. This site is designed, built, and audited by me — and held to the same standard I bring to client work.

What This Site Supports

  • Keyboard navigation throughout
  • Screen reader compatibility (tested with VoiceOver on macOS and iOS)
  • Skip to content link for keyboard users
  • ARIA landmarks, labels, and live regions
  • High contrast mode toggle
  • Text size toggle
  • Dyslexia-friendly font option
  • Reduced motion support (respects your system settings automatically)
  • WCAG 2.1 AA compliance, audited using axe-core 4.9.1 and WAVE by WebAIM

Known Limitations

The rotating hero headline on the homepage is paused automatically for users with reduced motion enabled in their system settings. If you have any other issues navigating this site, please let me know.

Get in Touch

If something on this site isn't working for you — a screen reader issue, a contrast problem, anything — I want to know about it. Contact me directly.
